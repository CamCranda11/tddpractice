Version 1.1

Files::
junit17.jar - a copy of junit-platform-console-standalone-1.7.0.jar
Util.java - a utility class containing
 * an implementation of the Fibonacci sequence
 * a scaffolding for finding repeated values in a sequence of integers
UtilTest.java - compilable JUnit test class for Util

To compile::
javac -cp junit17.jar:. *.java

To run the unit tests::
java -jar junit17.jar --class-path . --select-class UtilTest

To produce API documentation::
javadoc -cp junit17.jar:. -d docs *.java
